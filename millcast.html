      Assignment Part 1- Molding From A Negative

      1. Design an 3D ORIGINAL .stl file to use in the MDX40A.

      2. Mill the part using subtractive manufacturing using the MDX40A. You will mill in wax.

      3. Create a mold using Smooth-Sil 960.

      4. Cast a chocolate part.



      Assignment Part 2- Pewter Pendant

      1. Design an 2D ORIGINAL .stl file to use in the MDX40A. Include a sprue and vents for your pour.

      2. Mill the part using subtractive manufacturing using the MDX40A. It must be smaller than 2" by 2".

      3. Cast the part you milled in pewter to make a pendant.

      OR

      1. Use the laser to create a mold for your pewter pendant. You will include a sprue and vents for your pour.

      2. Use CorelDraw to engrave a 3" by 3" piece of MDF. You can see examples, One and Two

      OR

      1. You can use the MoldMax 60 from Smooth-On to create your mold. This will be done on Sunday, March 5th. You will have to create your positive and then we will make the negative using Smooth-On MoldMax 60.
